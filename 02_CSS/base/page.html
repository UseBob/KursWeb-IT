  • Должен присутсвовать цвет фона
  • Распологается слева
  • Блок навигации наполниет контентом достаточным, чтобы по высоте он начинал захватывать вторую статью,как показано в данном примере

Статья 1

Размер блока:
занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех статьях сделайте отступ снизу от соседней равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным ет, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.
Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом.
